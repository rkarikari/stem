🔬 Engineering Conclusions
System Integration

Handheld range is system-limited, not antenna-only limited. The complete RF chain must be considered for accurate performance prediction.

Impedance Matching

PA impedance match can swing realized EIRP by 5-10 dB, equivalent to 2-3x range variation.

🎯 Key Takeaways:
  • Band Dependency: VHF outperforms UHF only when the antenna-HT pairing is favorable
  • Context Matters: "Better antenna" is meaningless without specifying the radio
  • Measurement-Driven: All conclusions derived from field measurements, not theory
  • System Approach: Optimize the complete system, not individual components
💡 Recommendations
  1. For VHF-primary operation: SRH-770S delivers best performance (+6 dB vs stock)
  2. For UHF-primary operation: SRH-77CA is the clear choice (+8.6 dB vs stock)
  3. For dual-band balanced use: Consider application priorities and typical operating band
  4. Before purchasing: Verify compatibility with your specific HT model
