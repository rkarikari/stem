Welcome to RadioSport

RadioSport is a comprehensive STEM education program that integrates amateur radio, satellite communications, drone operations, artificial intelligence, renewable energy, and advanced software development. Students gain hands-on experience with cutting-edge technologies while developing critical thinking, problem-solving, and teamwork skills.

This handbook contains 14 comprehensive chapters covering everything from basic RF theory to advanced satellite operations and autonomous drone systems. Each chapter includes detailed technical explanations, practical exercises, code examples, and safety protocols.
