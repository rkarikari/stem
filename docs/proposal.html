ADDENDUM A1

HT Antenna System Analyzer

Interactive RF performance evaluation of handheld transceiver antenna systems. Analyzes efficiency, impedance matching, chassis coupling, and VHF/UHF radiation patterns from real field measurements.

Field Measurements VHF/UHF Interactive

ADDENDUM A2

Linear Transponder Frequency Planner

Real-time uplink/downlink frequency reference for LEO amateur satellites. Supports inverting and non-inverting transponders with 5 kHz channel raster, LSB/USB designations, and visual band sliders for Doppler planning.

Uplink/Downlink LSB/USB Doppler

ADDENDUM A3

Accra HF/VHF/UHF Coverage Planner

Terrain-aware propagation modelling using SRTM 90 m elevation data and Okumura-Hata / COST-231 clutter budgets. Adjustable power, antenna selection, beam heading, and multi-band switching with real-time map overlay.
