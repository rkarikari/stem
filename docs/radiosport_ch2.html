Tuning Engineering potential into Performance

RadioSport's curriculum integrates five knowledge streams‚ÄîAI, Radio, Satellites, Drones, and Coding‚Äîinto a comprehensive 9-month program. Students progress through foundation, integration, specialization, and mastery phases.

Five Knowledge Streams

Stream 1: Offline AI & Computational Thinking

Core Topics:

  • Classical AI: Search algorithms (A*, Dijkstra), heuristics, constraint satisfaction
  • Machine Learning: K-means, decision trees, na√Øve Bayes, random forests, SVMs
  • Neural Networks: Perceptrons, backpropagation, activation functions, offline training
  • AI for Radio: Propagation prediction, interference classification, signal optimization
  • Satellite Intelligence: Orbital prediction (SGP4/SDP4), Doppler correction, pass scheduling
  • Drone Autonomy: Path planning, obstacle avoidance, computer vision, mission optimization

Stream 2: Radio & RF Theory

Core Topics:

  • RF Fundamentals: Frequency, wavelength, power (dBm/dBW), impedance, SWR
  • Transmission Lines: Coax characteristics, velocity factor, loss, connectors
  • Antenna Theory: Radiation patterns, gain, polarization, feedpoint impedance
  • Antenna Types: Dipoles, verticals, Yagis, helical, log-periodic, satellite antennas
  • Propagation: Line-of-sight, ionospheric layers, tropospheric ducting, sporadic-E
  • Link Budget: TX power + gain - loss + RX gain = received signal
  • Digital Modes: AFSK1200, APRS, D-STAR, DMR, PSK31, FT8, SSTV
  • SDR Technology: I/Q sampling, demodulation, spectrum analysis

Stream 3: Amateur Radio Satellite Communications

Core Topics:

  • Orbital Mechanics: Kepler's laws, TLE format, orbital period calculations
  • Active Satellites: AO-91, AO-92, SO-50, ISS operations
  • Tracking Software: Gpredict, pass prediction, real-time Az/El
  • Doppler Compensation: Frequency shift calculations, automated tuning
  • Ground Stations: Crossed-Yagi design, rotator control, preamplifiers
  • Operating Procedures: FM vs SSB modes, uplink/downlink coordination

Stream 4: Drone Operations & Integration

Core Topics:

  • Aviation Fundamentals: Four forces, airfoil theory, multirotor dynamics
  • DJI Platforms: Phantom 4, Mavic Pro systems and specifications
  • Flight Planning: Mission software, waypoint navigation, area coverage
  • Radio Integration: MAVLink, telemetry transmission, payload mounting
  • Battery Management: LiPo characteristics, C-rating, charge/discharge cycles
  • Regulations: Airspace classes, altitude limits, VLOS requirements

Stream 5: Coding & Software Engineering

Core Topics:

  • Python: NumPy, SciPy, Matplotlib, PySerial, asyncio
  • C++: Real-time systems, memory management, templates, STL
  • RF Applications: Propagation simulators, antenna calculators, SDR control
  • Satellite Software: TLE parsing, SGP4 propagation, Doppler engines
  • Drone Programming: DroneKit-Python, MAVLink, autonomous missions
  • Signal Processing: FFT implementation, filter design, demodulation

Participant Requirements

Mandatory Requirements:

  • Attendance: Minimum 80% (29/36 weekly, 7/9 field days)
  • Donation: $15/month minimum (sliding scale available)
  • Documentation: Weekly logs, code submissions (GitHub), contact logging
  • Teamwork: Pairs or small groups (3-4) for complex projects
  • Safety: Zero tolerance for unsafe behavior
  • Equipment Care: Proper handling, storage, immediate damage reporting
  • Age: 13-19 years (11-12 with parent co-participation)
