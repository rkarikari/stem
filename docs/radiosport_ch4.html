VHF/UHF Line-of-Sight Propagation

📡 Radio Horizon and Line-of-Sight Propagation
VHF/UHF signals follow line-of-sight paths but extend beyond the optical horizon due to atmospheric refraction

Radio Horizon Formula

d (km) = 4.12 × [√h₁ + √h₂]

where h₁ and h₂ are antenna heights above ground in meters

Examples:

  • Base to mobile: h₁=25m, h₂=2m → d = 4.12 × [5 + 1.41] = 26.4 km
  • Mountain to valley: h₁=500m, h₂=10m → d = 4.12 × [22.36 + 3.16] = 105 km
  • Drone relay: h₁=100m, h₂=2m → d = 4.12 × [10 + 1.41] = 47 km

HF Ionospheric Propagation

🌍 Ionospheric Layers and HF Skywave Propagation
HF radio waves refract off ionospheric layers, enabling long-distance communication beyond line-of-sight. The F2 layer is most important for DX.

Ionospheric Layers

  • D Layer (60-90 km): Daytime only, absorbs HF (especially low bands)
  • E Layer (90-150 km): Regular and Sporadic-E propagation
  • F1 Layer (150-220 km): Daytime split of F layer
  • F2 Layer (220-400 km): Primary HF propagation, strongest layer

Key Concepts:

  • MUF (Maximum Usable Frequency): Highest frequency refracted back to Earth
  • LUF (Lowest Usable Frequency): Below this, D-layer absorption too high
  • Skip Zone: Region between ground wave and first sky wave return

Antenna Fundamentals

📐 Dipole Antenna Radiation Pattern
Half-wave dipole radiates maximum power broadside (perpendicular) to the antenna, with nulls off the ends
📡 Yagi-Uda Directional Antenna
Yagi-Uda antenna uses parasitic elements to create a highly directional beam with significant forward gain

Antenna Characteristics

Dipole (λ/2):

  • Impedance: 73Ω at resonance
  • Gain: 2.15 dBi
  • Pattern: Figure-8 (broadside maximum)
  • Bandwidth: 2-5% of center frequency

Quarter-Wave Vertical (λ/4):

  • Impedance: ~36Ω (requires matching)
  • Gain: 3-5 dBi (low angle radiation)
  • Requires: Ground plane (4 radials minimum)
  • Best for: DX, mobile operations

Link Budget Analysis

📊 Radio Link Budget Components
Link budget analysis predicts received signal strength by accounting for all gains and losses in the transmission path

Link Budget Equation

P_RX (dBm) = P_TX + G_TX - L_cable - L_path + G_RX - L_cable

Free-Space Path Loss:

FSPL (dB) = 32.45 + 20×log₁₀(f_MHz) + 20×log₁₀(d_km)

Example: 2m VHF at 50 km

  • TX power: +37 dBm (5W)
  • TX antenna gain: +6 dBi
  • TX cable loss: -2 dB
  • Path loss: -111 dB
  • RX antenna gain: +2 dBi
  • RX cable loss: -0.5 dB
  • Received: -68.5 dBm (excellent signal)

Digital Modes

Operating Procedures

VHF/UHF FM Simplex

Making First Contact:

  1. Listen first—ensure frequency is clear
  2. Call: "CQ CQ CQ, this is [your callsign]"
  3. Listen for response
  4. Exchange: Signal reports, location, name
  5. Keep transmissions brief and clear
  6. End: "73" (best regards) and sign off

Repeater Operation

🔄 VHF Repeater Operation
VHF repeaters extend communication range by receiving on one frequency and simultaneously retransmitting on another

Repeater Basics:

  • 2m offset: ±600 kHz
  • 70cm offset: ±5 MHz
  • CTCSS/PL tone: Sub-audible access tone (67-254 Hz)

Example: 146.940 MHz Repeater

  • Output (listen): 146.940 MHz
  • Input (transmit): 146.340 MHz (-600 kHz)
  • CTCSS: 103.5 Hz

Practical Exercises

Exercise 1: Range vs Height Test

  1. Set up base station at known elevation
  2. Mobile station drives outward, recording distances
  3. Note maximum reliable communication range
  4. Compare measured to theoretical radio horizon
  5. Identify terrain effects (hills, buildings)

Exercise 2: SWR Measurement

  1. Connect SWR meter between radio and antenna
  2. Set radio to low power (5-10W)
  3. Sweep frequency across band in 100 kHz steps
  4. Record SWR at each frequency
  5. Plot SWR curve, identify resonant point
  6. Target: SWR < 2:1 across operating range
