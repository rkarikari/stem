Aviation Fundamentals for Multirotors

Four Forces of Flight

Quadcopter Flight Dynamics - Four Forces
Four fundamental forces acting on a multirotor drone in forward flight

Understanding Multirotor Dynamics:

  • Lift: Upward force from rotor thrust (must equal weight for hover)
  • Weight: Downward gravitational force (mass √ó 9.8 m/s¬≤)
  • Thrust: Forward component when drone tilts (controlled by differential rotor speeds)
  • Drag: Air resistance opposing motion (increases with speed)

Flight Control Methods:

  • Hover: All rotors equal speed, total thrust = weight
  • Climb: Increase all rotor speeds equally (thrust > weight)
  • Descend: Decrease all rotor speeds (thrust < weight)
  • Forward: Tilt forward by speeding up rear rotors, slowing front
  • Yaw: Opposite rotor pairs spin at different speeds (torque imbalance)

Drone-Radio Integration Applications

Application 1: Elevated Antenna Platform

Radio Horizon Extension with Drone Altitude
Dramatic range extension achieved by elevating antenna to 100m altitude

Concept: Extending VHF/UHF Range

Theory: Radio horizon = 4.12 √ó ‚àöh kilometers

Hardware Setup:

  • Handheld VHF transceiver (200g with battery)
  • Lightweight dipole antenna (20g)
  • PTT relay (RC-controlled or autonomous)
  • 3D-printed mounting cradle

Application 2: Propagation Mapping

RF Coverage Mapping Flight Pattern
Autonomous grid pattern for 3D RF coverage mapping with signal strength measurement at each waypoint

Objective: Create 3D RF Coverage Map

Equipment:

  • Drone with GPS and data logger
  • RF power meter or SDR receiver (RTL-SDR, 50g)
  • VHF/UHF antenna
  • Storage for telemetry data

Procedure:

  1. Ground station transmits continuous carrier
  2. Drone flies programmed grid pattern at multiple altitudes
  3. Records signal strength (RSSI) at each waypoint
  4. GPS tags each measurement
  5. Downloads data, creates heatmap

Analysis:

  • Identify terrain shadowing effects
  • Measure antenna pattern in real environment
  • Determine optimal repeater placement
  • Observe altitude effects on range
  • Validate propagation models

Application 3: Airborne Relay Station

Cross-Band Repeater Configuration
Airborne cross-band repeater extends communication range between ground stations

Temporary Repeater for Field Operations

Concept: Drone carries dual-band radio configured as cross-band repeater

Hardware:

  • Dual-band transceiver (Yaesu FT-8900R or similar)
  • Dual-band antenna system
  • Duplexer or isolated antennas
  • 12V power from drone battery (regulated)
  • VOX or COR for automatic triggering

Configuration:

  • Receive: 146.520 MHz (2m calling freq)
  • Transmit: 446.000 MHz (70cm)
  • Altitude: 100m AGL
  • Coverage: ~50 km radius

Applications:

  • Emergency communications
  • Search and rescue operations
  • Field day extended coverage
  • Special event communications
  • RF propagation demonstrations

Safety Considerations:

  • RF isolation to prevent desensitization
  • Power consumption monitoring (5-15W continuous)
  • Reduced flight time (10-15 min under load)
  • Frequency coordination required
  • Timeout timer (5-minute max transmission)
